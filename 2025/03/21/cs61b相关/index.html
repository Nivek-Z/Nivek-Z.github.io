Proj1A: LinkedListDeque61B

用双向链表实现动态列表

Proj1B: ArrayDeque61B

用数组实现列表

设计要点

  • 用 front 和 back 分别标记头和尾，size 标记列表长度
  • 总体设计和双向链表一样，也是循环的

示例说明

  • 假设初始数组大小为 8，front 初始为 0
  • 如果数组索引 0 处有一个 "zero"，调用 addFirst("first") 方法时，会在数组索引 7 的位置加入一个 "first"
  • 此时让 front 减去 1，front 就会变为 -1，但在原始数组中 -1 无法访问到数组的末尾
  • 通过取模运算（-1 % 数组大小），得到索引 7，因此 front 变为 7
  • 列表的顺序与数组的索引顺序无关
  • 从 front 开始遍历到 front + size 即可打印出列表中所有元素
